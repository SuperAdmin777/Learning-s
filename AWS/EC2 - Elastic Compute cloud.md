### Traditional Server (EC2)

Imagine you own a restaurant.

- You rent a building.
- You buy the kitchen equipment.
- You hire cooks.
- You keep the lights on all day.
- Even if **no customers come**, you still pay for everything.

This is like **Amazon EC2**.

You are responsible for:

- Setting up the server
- Installing software
- Updating the operating system
- Scaling when more users arrive
- Paying while the server is running
